# Date Calculator
<p align="center">
  <img width="700" src="_layout_pics/dateCalculator.png">
</p>

A simple date calculator written in HTML5, JavaScript ES6 which outputs how many years and days are passed from the entered date until now. <br>
The input field has a RegEx pattern which validates whether the entered date is in the format 'yyyy-mm-dd'.<br>
For the output is here used the "dateFns" library.

# MyQuiz
<p align="center">
  <img width="700" src="_layout_pics/myQuiz.png">
</p>

A simple quiz with 4 questions and 2 possible answers. <br>
The right answers are saved in an array and used to compare to the user's choice.<br>
The window methods are used in the application (scrollTo(), setInterval(), clearInterval()).
